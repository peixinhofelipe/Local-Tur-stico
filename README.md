Site de Turismo

Este projeto foi desenvolvido como uma atividade para praticar HTML e CSS. O objetivo foi criar uma página simples de turismo, utilizando os conhecimentos aprendidos durante os estudos.

Tecnologias utilizadas
HTML5
CSS
O que foi feito
Estrutura da página com HTML.
Estilização com CSS.
Organização do conteúdo em seções.
Inserção de imagens e textos.
Uso de cores, fontes e espaçamentos.
Como abrir o projeto

Basta abrir o arquivo index.html em um navegador para visualizar o site.

Objetivo

Este projeto foi feito para colocar em prática os conceitos básicos de HTML e CSS, ajudando no aprendizado da criação e estilização de páginas web.
